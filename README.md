# Meta Quest XR Latency Experiment

This project is a Unity-based XR experiment for Meta Quest, developed in collaboration with a PhD researcher. The experiment was designed to simulate latency in a mixed-reality environment and study its effect on user experience and cybersickness.

## Features

* Unity XR application targeting Meta Quest.
* Mixed-reality experiment setup for latency simulation.
* Integration of Meta Shared Spatial Anchors for persistent spatial alignment.
* Use of Meta Colocation Discovery to support shared multi-user mixed-reality spaces.
* Co-located spatial alignment so users could interact within the same physical environment.
* Experimental setup for measuring cybersickness responses under different latency conditions.

## Project Focus

The goal of this project was to support a controlled XR research experiment by building the technical foundation for shared mixed-reality interaction. The project focused on spatial alignment, multi-user colocation, latency simulation, and reliable experimental setup on Meta Quest hardware.

## Technologies Used

* Unity
* C#
* Meta Quest
* Meta Mixed Reality APIs
* Shared Spatial Anchors
* Colocation Discovery
* XR interaction systems
